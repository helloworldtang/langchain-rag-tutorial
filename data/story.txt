Python 是一种广泛使用的高级编程语言。
它由 Guido van Rossum 在 20 世纪 90 年代初设计。
Python 的设计哲学强调代码的可读性和简洁的语法。
与 C++ 或 Java 不同，Python 使用缩进来定义代码块，而不是大括号。
这使得 Python 代码非常易于阅读和维护。
Python 拥有庞大的标准库和第三方库生态系统，可以处理 Web 开发、数据分析、人工智能、自动化运维等多种任务。
Python 是人工智能和机器学习领域最受欢迎的编程语言之一。
像 TensorFlow、PyTorch、Scikit-learn 等强大的 AI 库都优先支持 Python。
LlamaIndex 是一个强大的数据框架，用于构建大语言模型（LLM）应用程序。
它允许你将私有数据（如 PDF、Notion 笔记、SQL 数据库、本地文本文件等）连接到 LLM 上。
LlamaIndex 的核心流程是检索增强生成（RAG）。
LlamaIndex 提供了 100 多种数据连接器，支持各种数据源。
LlamaIndex 支持多种索引类型，如向量存储索引、摘要索引、关键词索引。
LlamaIndex 与许多向量数据库集成，如 ChromaDB、Pinecone、Weaviate。
LlamaIndex 与许多 LLM 集成，如 OpenAI、Anthropic、Cohere。
LlamaIndex 是生产就绪的，内置可观测性、评估和追踪功能。
